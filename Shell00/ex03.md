# Shell00 — Exercise 03 : SSH me

## Objectif

Générer une paire de clés SSH et soumettre **uniquement la clé publique**.

---

## Fichier à rendre

```
ex03/
└── id_rsa.pub
```

---

## Étapes

### 1. Générer la paire de clés SSH

```sh
ssh-keygen -t rsa
```

Quand le terminal te pose des questions :

```
Enter file in which to save the key (~/.ssh/id_rsa):  ← Entrée (défaut)
Enter passphrase (empty for no passphrase):           ← Entrée (pas de passphrase)
Enter same passphrase again:                          ← Entrée
```

Deux fichiers sont créés dans `~/.ssh/` :

| Fichier | Rôle |
|---------|------|
| `id_rsa` | Clé **privée** — ne jamais partager |
| `id_rsa.pub` | Clé **publique** — c'est celle qu'on rend |

---

### 2. Copier la clé publique dans le dossier de rendu

```sh
cp ~/.ssh/id_rsa.pub ~/rendu/ex03/id_rsa.pub
```

---

### 3. Vérifier le contenu du fichier

```sh
cat ex03/id_rsa.pub
```

Le fichier doit commencer par :

```
ssh-rsa AAAAB3Nza...
```

---

## Ce que le correcteur vérifie

- Le fichier `id_rsa.pub` est présent
- C'est bien une clé publique SSH valide (format `ssh-rsa`)
- Aucune passphrase n'a été utilisée

---

## À ne pas faire

```sh
# ❌ Ne jamais rendre la clé privée
id_rsa        # INTERDIT — clé privée

# ✅ Rendre uniquement la clé publique
id_rsa.pub    # CORRECT
```
